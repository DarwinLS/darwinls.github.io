Clinical Evidence Synthesis Engine

A live Real-Time RAG Pipeline that aggregates and synthesizes clinical trials on demand. Unlike generic chatbots, this engine uses a two-stage "Curator-Synthesizer" architecture: first performing semantic filtering to reject irrelevant studies, then using lazy evaluation to extract high-fidelity dosage and safety protocols from raw text.
